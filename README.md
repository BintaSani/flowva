# Flowva

A React + TypeScript application built with Vite.  
Includes authentication, onboarding, referral rewards, and point redemption.

---

## Tech Stack

- React
- TypeScript
- Vite
- Supabase (Auth + Database)
- Tailwind CSS

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/flowva.git
cd flowva

### 2. insstall dependencies
npm install

### 3. create .env file
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

### 4. run app
npm run dev

open http://localhost:5173

### 5. live site

https://flowva-green.vercel.app/
```
